# 🐾 Neko Mahjong

**Neko Mahjong** é um jogo de puzzle mobile desenvolvido em **Godot 4.x**, focado em oferecer uma experiência relaxante e visualmente encantadora para amantes de gatos. 

O projeto é otimizado especificamente para dispositivos de alta performance, como o **Samsung Galaxy S25**, garantindo que as artes das peças sejam o destaque absoluto da tela.

## ✨ Diferenciais do Projeto
* **Sistema de 4 Slots**: Mecânica de inventário dinâmico que exige estratégia para gerenciar espaço.
* **Layouts Portrait-First**: Tabuleiros desenhados verticalmente para maximizar a escala das peças em smartphones.
* **Regra 90/10**: Sistema de colisão preciso para determinar bloqueios de peças superiores.
* **Arte Nítida**: Sprites de gatos que ocupam 90% da face das peças para máxima visibilidade.

## 🛠️ Tecnologias
* **Engine**: Godot 4.6+
* **Linguagem**: GDScript
* **Target**: Android (Galaxy S25)
